# Medical Abstract Classification
A natural language processing project that classifies biomedical abstracts into disease categories using transformer-based models.

## Categories

- Neoplasms
- Digestive System Diseases
- Nervous System Diseases
- Cardiovascular Diseases
- General Pathological Conditions

## Models

### Baseline
- TF-IDF
- Logistic Regression

### Deep Learning
- DistilBERT
- Hugging Face Transformers
- PyTorch

## Results

| Model | Accuracy |
|---------|---------|
| TF-IDF + Logistic Regression | 55.5% |
| DistilBERT | Training in progress |
